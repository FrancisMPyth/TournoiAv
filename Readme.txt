# Projet 4 OpenVlassRoom 

Ce programme à été créé pour le projet 4, qui consistait à créer un programme de jeu d'échec répondant a un cahier des charges.
Le but de ce projet est de créer un logiciel de gestion de tournois d'échecs qui fonctionne hors ligne. 8 joueurs minimum participent à un tournoi 
qui se compose de 4 rounds et chaque round comprend 4 matchs. 
Les organisateurs vont pouvoir créer les joueurs (8 au minimum) , encoder les tournois et sortir des rapports sur le déroulement des tournois. 
Chaque tournoi pourra être arrêté temporairement à tout moment. 
Les résultats de chaques matchs pourront être enrigistrés à la fin de chaque matchs.

## Exigences du Système

- Python 3.10 et supérieur
- Visual Studio Code 1.84.2

## Installation

1. Installer Python
2. Installer V.S.C
3. Itégrer git Bash dans V.S.C
4. Cloner le projet : https://github.com/FrancisMPyth/TournoiAv.git
5. Installer les dépendances : `pip install -r requirements.txt`

## Utilisation

exécuter main.py
